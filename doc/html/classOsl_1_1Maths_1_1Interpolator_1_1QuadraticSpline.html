Detailed Description

Class to construct a piecewise Quadratic Spline interpolator from real data.

Principle

This class defines a one dimensional piecewise quadratic spline interpolator. Let a set of evaluation points \((x_k)_{k\in[\vert0;N\vert]}\) of size \(N+1\) being in a strictly increasing order:

\[ x_0<x_1<\cdots<x_{n-1}<x_n \]

The piecewise quadratic spline interpolation function at index \(k\) is defined by the second order polynomial:

\[ f_k(x)=a_k(x-x_k)^2+b_k(x-x_k)+c_k,\quad \forall x\in[x_k; x_{k+1}[\quad ; \quad\forall k\in[\vert0;N-1\vert] \]

providing a set of \(N\) interpolation functions \((f_k)_{k\in[\vert0;N-1\vert]}\) with their corresponding sets of coefficients \((a_k, b_k, c_k)_{k\in[\vert0;N-1\vert]}\).

Computation of the interpolator coefficients

The piecewise quadratic spline coefficients are estimated by ensuring continuity of the interpolation function and its first derivative at each node, that is:

\[ \begin{array}{rcl} f_k(x_k) & = & y_k\\ f_{k+1}(x_{k+1}) & = & f_k(x_{k+1})\\ f'_{k+1}(x_{k+1}) & = & f'_k(x_{k+1}) \\ \end{array} \]

with \(y_k\) being the value of the function to be interpolated at the given point \(x_k\).

However, this system only contains \(3N-1\) equations whereas \(3N\) equations are needed to estimate the \(3N\) coefficients of the interpolation functions. To circumvent this problem, one can choose to consider the first or the last interpolation function to be linear. This offers two boundary conditions for estimating the coefficients of the piecewise quadratic spline:

  • The first interpolation function is considered as linear, that is \(a_0=0\). In this case the coefficients of the quadratic spline are found iteratively by solving the previous system:

    \[ \left\{\begin{array}{rcl} c_k & = & y_k\\ b_k & = & b_{k-1}+2a_{k-1}(x_k-x_{k-1})\\ a_k & = & \dfrac{1}{x_{k+1}-x_{k}} \left(\dfrac{y_{k+1}-y_{k}}{x_{k+1}-x_{k}}-b_k\right) \end{array}\right.\quad\forall k\in[\vert1;N-1\vert] \]

    with the initial condition:

    \[ \left\{\begin{array}{rcl} c_0 & = & y_0\\ b_0 & = & \dfrac{y_{1}-y_{0}}{x_{1}-x_{0}}\\ a_0 & = & 0 \end{array}\right.\quad\mbox{ for }k=0 \]

  • The last interpolation function is considered as linear, that is \(a_{n-1}=0\). In this case the coefficients of the quadratic spline are found iteratively by solving the previous system:

    \[ \left\{\begin{array}{rcl} c_k & = & y_k\\ a_k & = & \dfrac{1}{x_{k+1}-x_{k}} \left(b_{k+1}-\dfrac{y_{k+1}-y_{k}}{x_{k+1}-x_{k}}\right)\\ b_k & = & b_{k+1}-2a_k(x_{k+1}-x_k) \end{array}\right.\quad\forall k\in[\vert N-2;0\vert] \]

    with the initial condition:

    \[ \left\{\begin{array}{rcl} c_{n-1} & = & y_{n-1}\\ a_{n-1} & = & 0\\ b_{n-1} & = & \dfrac{y_{n}-y_{n-1}}{x_{n}-x_{n-1}} \end{array}\right.\quad\mbox{ for }k=n-1 \]

◆ search_index_for_interpolation()

std::size_t Osl::Maths::Interpolator::QuadraticSpline::search_index_for_interpolation ( const double &  xeval)

Search the index for spline interpolation.

This function makes use of binary search to find the first index \(i\) of reference axis \(x[i]\) (in strictly increasing order) such that, for the search value \(x_{eval}\):

\[ x_{eval} \geq x[i] \]

Parameters
[in]xevalthe value for which the index is searched.
Returns
The index to evaluate the spline function at xeval.
Note
1. If \(x_{eval}\geq\underset{i}{\max}(x[i])\), last index minus 1 is returned.
2. If \(x_{eval}\leq\underset{i}{\min}(x[i])\), index 0 is returned.
