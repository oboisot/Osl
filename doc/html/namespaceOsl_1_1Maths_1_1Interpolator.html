Detailed Description

This is the Osl::Maths::Interpolator namespace which provides a set of interpolator classes.

Enumerator

Create a cubic spline interpolator with "natural" boundary conditions.

Create a cubic spline interpolator with first and last polynomials being quadratic polynomials.

Create a cubic spline interpolator with "not-a-knot" boundary conditions.
