Function to compare two real double precision floating points.

This function compares two double precision floating points \(x\) and \(y\) against the machine resolution for double \(res\):

\[ \left\vert x-y\right\vert\leq res \]

with the machine resolution defined in Osl::Constants::n_machine_res. When this equality holds, the two numbers are considered as almost equal.

Parameters
[in]x,ythe two numbers to be compared.
Returns
true if x and y are equal in a machine sense, false otherwise.

Function to compare two complex double precision floating points.

This function compares two complex double precision floating points \(x\) and \(y\) against the machine resolution for double \(res\). The comparison is made on both real and imaginary parts of the two complex numbers as follows:

\[ \left\vert\mathcal{R}(x)-\mathcal{R}(y)\right\vert\leq res \quad\mbox{and}\quad \left\vert\mathcal{I}(x)-\mathcal{I}(y)\right\vert\leq res \]

with the machine resolution defined in Osl::Constants::n_machine_res. When these equality holds, the two complex numbers are considered as almost equal.

Parameters
[in]x,ythe two complex numbers to be compared.
Returns
true if x and y are equal in a machine sense, false otherwise.
See also
bool almost_equal(double x, double y)
