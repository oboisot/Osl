Detailed Description

Class to construct a piecewise Linear Spline interpolator from complex data.

Principle

This class defines a one dimensional piecewise linear spline interpolator. Let a set of evaluation points \((x_k)_{k\in[\vert0;N\vert]}\) of size \(N+1\) being in a strictly increasing order:

\[ x_0<x_1<\cdots<x_{n-1}<x_n \]

The piecewise Linear Spline interpolation function at index \(k\) is defined by the first order polynomial:

\[ f_k(x)=a_k(x-x_k) + b_k,\quad\forall x\in[x_k;x_{k+1}[\quad ; \quad\forall k\in[\vert0;N-1\vert] \]

providing a set of \(N\) interpolation functions \((f_k)_{k\in[\vert0;N-1\vert]}\) with their corresponding coefficients \((a_k, b_k)_{k\in[\vert0;N-1\vert]}\).

Computation of the interpolator coefficients

Coefficients are determined by ensuring the continuity of the interpolation function at each interval nodes:

\[ \begin{array}{rcl} f_k(x_k) & = & y_k\\ f_k(x_{k+1}) & = & y_{k+1}\\ \end{array} \]

Solving this system results in the evaluation of the coefficients for the piecewise linear interpolation functions \(f_k\):

\[ \left\{\begin{array}{rcl} a_k & = & \dfrac{y_{k+1}-y_k}{x_{k+1}-x_k} \\ b_k & = & y_k \end{array}\right., \quad\forall k\in[\vert0;N-1\vert] \]

LinearSpline a linear spline interpolator class for real data.

◆ search_index_for_interpolation()

std::size_t Osl::Maths::Interpolator::ComplexLinearSpline::search_index_for_interpolation ( const double &  xeval)

Search the index for spline interpolation.

This function makes use of binary search to find the first index \(i\) of reference axis \(x[i]\) (in strictly increasing order) such that, for the search value \(x_{eval}\):

\[ x_{eval} \geq x[i] \]

Parameters
[in]xevalthe value for which the index is searched.
Returns
The index to evaluate the spline function at xeval.
Note
1. If \(x_{eval}\geq\underset{i}{\max}(x[i])\), last index minus 1 is returned.
2. If \(x_{eval}\leq\underset{i}{\min}(x[i])\), index 0 is returned.
